Geolocation API

O que é

A API de geolocalização do HTML5 é usada para recuperar a posição geofísica do usuário.

Como essa informação é privada, a posição não é recuperada a menos que o usuário conceda permissão.

As requisições são feitas via HTTPS usando o método POST.


A acurácia é maior em dispositivos que possuem GPS.

O que é

A localização e o raio de 'acerto' são baseados em informações enviadas por torres de celular e pontos de WiFi.

Para viasualizar a localização no mapa, é necessária a utilização de algum serviço, como o Google Maps.

Navegadores

Se o site estiver em um ambiente não-seguro (HTTP), a requisição para solicitar a localização do usuário não funcionará.

Usando a API

Principais métodos:

  • getCurrentPosition()
  • watchPosition()
  • clearWatch()

Usando a API

getCurrentPosition()

É o método usado para retornar a posição do usuário propriamente dita. O método retorna a latitude, a longitude e as propriedades de acurácia.

Outras propriedades podem ser retornadas se estiverem disponíveis.

Usando a API

getCurrentPosition(): valores retornados

  • coords.latitude
  • coords.longitude
  • coords.accuracy
  • coords.altitude
  • coords.altitudeAccuracy
  • coords.speed
  • coords.timestamp

Usando a API

watchPosition()

Método que retorna a posição atual do usuário e continua retornando a posição atualizada a medida que o usuário se move.

clearWatch()

Interrompe o método watchPosition()

Exemplos

The Geolocation API returns a location and accuracy radius based on information about cell towers and WiFi nodes that the mobile client can detect. This document describes the protocol used to send this data to the server and to return a response to the client.

Gamepad API

O que é?

A Gamepad API possibilita a conexão de controles de video games no computador.

  • Detecta conexão e desconexão de eventos.
  • Reconhece múltiplos gamepads
  • Inspeciona gamepads conectados e reconhece quantos botões eles possuem e qual é o estado de cada um

Navegadores

Não é suportada pelo IE nem pelo Opera.
